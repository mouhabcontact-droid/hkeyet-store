# Optimisation SEO - Librairie en Ligne Tunisie

## Vue d'ensemble

Votre site HKEYET Publishing a été optimisé pour être le **#1 en Tunisie** pour la vente de livres en ligne. Toutes les pages sont optimisées en français pour le marché tunisien.

## Fonctionnalités SEO Implémentées

### 1. Meta Tags Optimisés

Chaque page contient des meta tags complets en français :
- **Titre** : Optimisé avec mots-clés principaux "librairie tunisie", "acheter livre en ligne tunisie"
- **Description** : 160 caractères max, avec appel à l'action
- **Mots-clés** : Liste complète de termes de recherche pertinents
- **Langue** : `lang="fr"` pour cibler le marché francophone tunisien

### 2. Données Structurées (Schema.org)

Implémentation de JSON-LD pour :
- **BookStore** : Informations sur la librairie
- **Book** : Chaque livre avec prix, disponibilité, auteur, ISBN
- **Organization** : HKEYET Publishing avec coordonnées Tunisie
- **SearchAction** : Recherche de livres optimisée pour Google

### 3. Balises Open Graph & Twitter Cards

Pour un meilleur partage sur les réseaux sociaux :
- Images optimisées (1200x630px)
- Titres et descriptions personnalisés
- Locale : `fr_TN` (Français Tunisie)
- Type de contenu approprié (book, website)

### 4. Optimisation par Page

#### Page d'accueil
- **Titre** : "Librairie en Ligne Tunisie #1 | HKEYET Publishing | Achat Livres & Ebooks"
- **Mots-clés principaux** : librairie tunisie, acheter livre en ligne tunisie, livres français tunisie

#### Pages de livres individuelles
- Titre unique pour chaque livre : "[Titre du livre] - [Auteur] | Librairie en Ligne Tunisie"
- Description basée sur le contenu du livre
- Schema.org Product avec prix en TND
- Mots-clés dynamiques basés sur le genre, l'auteur, le titre

#### Catalogue de livres
- "Tous les Livres | Librairie en Ligne Tunisie | Catalogue Complet"
- Optimisé pour recherches de livres générales

#### Ebooks
- "Ebooks en Ligne Tunisie | Livres Numériques à Télécharger"
- Ciblage du marché des livres numériques

#### Auteurs
- "Nos Auteurs | Écrivains et Auteurs Tunisiens et Internationaux"
- Liaison avec pages de livres individuelles

### 5. Géolocalisation Tunisie

- `geo.region` : TN
- `geo.placename` : Tunisia
- Coordonnées GPS : Tunis (36.8065, 10.1815)
- Langue : Français avec alternative Arabe

### 6. Fichiers SEO Essentiels

#### robots.txt
- Autorise l'indexation des pages publiques
- Bloque les pages privées (admin, dashboard, checkout)
- Indique l'emplacement du sitemap

#### sitemap.xml (Statique + Dynamique)
- Sitemap statique pour les pages principales
- Edge Function `/functions/v1/sitemap` pour génération dynamique
- Inclut toutes les pages de livres avec images
- Mise à jour automatique avec nouveaux livres
- Fréquence de mise à jour optimisée

### 7. Hreflang Tags

Configuration multilingue pour Tunisie :
- `fr-tn` : Français (Tunisie) - Principal
- `ar-tn` : Arabe (Tunisie) - Alternatif
- `x-default` : Version par défaut

### 8. Performance SEO

- Canonical URLs pour éviter le contenu dupliqué
- Meta robots optimisés : `index, follow, max-image-preview:large`
- Images optimisées avec alt text
- Temps de chargement rapide
